# Staying-alive
A wellness app where users care for a virtual pet that thrives based on their real-life
habits (eg: eating meals, staying hydrated, exercising and completing to-dos).
Each healthy action logged by the user directly supports the pet’s well-being, turning
self-care into a motivating, interactive experience.

---

## Installation and Usage

### Requirements

- Java 17+

> **Note:** This project does not enforce a specific Java version via build tools.

### Steps

1. Clone repository:

```
git clone https://github.com/Ugnytea/Staying-alive.git
```

2. Navigate to the source directory and run the application:
```
cd Staying-alive/src
java Main.java
```

---

## Features

- Track your sleep, nutrition, hydration, exercise and to-dos.
- Monitor your pet’s well-being and track your own self-care habits at the same time.
